## Overview

The `Colonia` model represents land colonies with property information. Each colony can span multiple districts and has auto-generated unique codes. The model includes state management and validation to ensure data integrity.

## Fields

<ParamField path="id" type="AutoField" required>
  Primary key automatically generated
</ParamField>

<ParamField path="nombre" type="string" required>
  Name of the colony
  - **Max length**: 250 characters
  - **Unique**: Yes
  - **Indexed**: Yes (for faster queries)
</ParamField>

<ParamField path="finca_matriz" type="string">
  Matrix farm identifier
  - **Max length**: 100 characters
  - **Nullable**: Yes
</ParamField>

<ParamField path="padron_matriz" type="string">
  Matrix property registry number
  - **Max length**: 100 characters
  - **Nullable**: Yes
</ParamField>

<ParamField path="distritos" type="ManyToManyField" required>
  Districts where the colony is located
  - **Related to**: `Distrito` model
  - **Related name**: `colonias`
  - **Validation**: Must have at least one distrito associated
</ParamField>

<ParamField path="estado" type="string" required>
  Current state of the colony
  - **Choices**:
    - `activo`: Active
    - `inactivo`: Inactive
  - **Default**: `activo`
  - **Max length**: 20 characters
</ParamField>

<ParamField path="codigo" type="PositiveInteger">
  Unique numeric code for the colony
  - **Unique**: Yes
  - **Nullable**: Yes (auto-generated on save)
  - **Auto-generated**: First available number starting from 1
</ParamField>

## Methods

### `save()`

Overridden to auto-generate the `codigo` field if not provided.

```python
def save(self, *args, **kwargs):
    if not self.codigo:
        codigos_existentes = list(
            self.__class__.objects.exclude(codigo__isnull=True)
            .values_list("codigo", flat=True)
        )
        nuevo_codigo = 1
        while nuevo_codigo in codigos_existentes:
            nuevo_codigo += 1
        self.codigo = nuevo_codigo
    super().save(*args, **kwargs)
```

**Logic**: Finds the first available positive integer not already used by existing colonies.

### `clean()`

Validates that the colony has at least one distrito associated.

```python
def clean(self):
    if self.pk:
        if self.distritos.count() == 0:
            raise ValidationError(
                "La colonia debe estar asociada a al menos un distrito."
            )
```

<Warning>
This validation only runs when the colony already exists (has a primary key). It cannot validate on initial creation before the many-to-many relationship is saved.
</Warning>

### `__str__()`

Returns the string representation of the colony.

```python
def __str__(self):
    return self.nombre
```

**Returns**: Colony name

## Relationships

### Many-to-Many
- **distritos**: Districts where this colony is located

### Reverse Relationships
- **solicitudes**: Survey requests for this colony (via `Solicitud.colonia` ForeignKey)
- **relevamientos**: Surveys conducted for this colony (via `Relevamiento.colonia` ForeignKey)

## Constraints

### unique_together

```python
class Meta:
```

The colony name must be globally unique across the system.

## Validation Rules

### Distrito Association
A colony **must** be associated with at least one distrito. This is enforced by the `clean()` method.

```python
# This will raise ValidationError
colonia = Colonia.objects.create(nombre="Test Colony")
colonia.clean()  # Raises: "La colonia debe estar asociada a al menos un distrito."
```

## Complete Model Definition

```python
from django.db import models
from django.core.exceptions import ValidationError

class Colonia(models.Model):
    ESTADO_CHOICES = [
        ("activo", "Activo"), 
        ("inactivo", "Inactivo")
    ]

    nombre = models.CharField(
        max_length=250, 
        db_index=True
    )
    finca_matriz = models.CharField(
        max_length=100, 
        blank=True, 
        null=True
    )
    padron_matriz = models.CharField(
        max_length=100, 
        blank=True, 
        null=True
    )
    distritos = models.ManyToManyField(
        Distrito, 
        related_name="colonias"
    )
    estado = models.CharField(
        max_length=20, 
        choices=ESTADO_CHOICES, 
        default="activo"
    )
    codigo = models.PositiveIntegerField(
        blank=True, 
        null=True, 
        unique=True
    )

    class Meta:
        unique_together = ("nombre",)
        verbose_name = "Colonia"
        verbose_name_plural = "Colonias"
        ordering = ["nombre"]

    def save(self, *args, **kwargs):
        if not self.codigo:
            codigos_existentes = list(
                self.__class__.objects.exclude(codigo__isnull=True)
                .values_list("codigo", flat=True)
            )
            nuevo_codigo = 1
            while nuevo_codigo in codigos_existentes:
                nuevo_codigo += 1
            self.codigo = nuevo_codigo
        super().save(*args, **kwargs)

    def clean(self):
        if self.pk:
            if self.distritos.count() == 0:
                raise ValidationError(
                    "La colonia debe estar asociada a al menos un distrito."
                )

    def __str__(self):
        return self.nombre
```

## Usage Example

```python
from core.models import Colonia, Distrito, Departamento
from django.core.exceptions import ValidationError

# Get districts
central = Departamento.objects.get(nombre='Central')
asuncion = Distrito.objects.get(nombre='Asunción', departamento=central)
lambare = Distrito.objects.get(nombre='Lambaré', departamento=central)

# Create a colony
colonia = Colonia.objects.create(
    nombre='Colonia Ejemplo',
    finca_matriz='FM-12345',
    padron_matriz='PM-67890',
    estado='activo'
)

# Associate with districts (MUST do this!)
colonia.distritos.add(asuncion, lambare)

# Validate
try:
    colonia.clean()
    print("Colony is valid")
except ValidationError as e:
    print(f"Validation error: {e}")

# Query colonies by distrito
colonias_asuncion = Colonia.objects.filter(
    distritos=asuncion,
    estado='activo'
)

# Get all districts for a colony
for distrito in colonia.distritos.all():
    print(f"- {distrito.nombre} ({distrito.departamento.nombre})")
```

## State Management

### Active vs Inactive

```python
# Activate a colony
colonia.estado = 'activo'
colonia.save()

# Deactivate a colony
colonia.estado = 'inactivo'
colonia.save()

# Query only active colonies
colonias_activas = Colonia.objects.filter(estado='activo')

# Query inactive colonies
colonias_inactivas = Colonia.objects.filter(estado='inactivo')
```

## Query Examples

### Get colonies spanning multiple districts
```python
from django.db.models import Count

colonias_multidistrito = Colonia.objects.annotate(
    num_distritos=Count('distritos')
).filter(num_distritos__gt=1)

for colonia in colonias_multidistrito:
    print(f"{colonia.nombre}: {colonia.num_distritos} distritos")
```

### Find colonies by department
```python
colonias_central = Colonia.objects.filter(
    distritos__departamento__nombre='Central'
).distinct()
```

### Get colonies with survey requests
```python
colonias_con_solicitudes = Colonia.objects.filter(
    solicitudes__isnull=False
).distinct().prefetch_related('solicitudes')

for colonia in colonias_con_solicitudes:
    print(f"{colonia.nombre}: {colonia.solicitudes.count()} solicitudes")
```

### Complex query: Active colonies with pending surveys
```python
colonias = Colonia.objects.filter(
    estado='activo',
    solicitudes__estado='pendiente'
).distinct().select_related(
    'distritos__departamento'
)
```
